GitHub部署指南

如何在GitHub上配置您的网站和图片

1. 创建GitHub仓库

登录GitHub，点击右上角"+"号，选择"New repository"

仓库名称格式: username.github.io (username替换为您的GitHub用户名)

选择公开(Public)，勾选"Add a README file"

2. 上传网站文件

点击"Add file" -> "Upload files"

将您的HTML文件拖拽到上传区域(可命名为index.html)

创建文件夹(如images)并上传您的图片资源

3. 更新图片路径

在HTML代码中，将图片路径更新为相对路径，例如:

将: https://placehold.co/600x400/1e90ff/white?text=每日委托

改为: images/daily-commission.jpg

4. 启用GitHub Pages

进入仓库的"Settings" -> "Pages"

在"Branch"下选择"main"分支，文件夹选择"/root"

点击"Save"，等待几分钟后您的网站就上线了

5. 访问您的网站

您的网站地址将是: https://username.github.io

将username替换为您的GitHub用户名
